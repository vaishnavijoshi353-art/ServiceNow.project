# HRONE – Automated Employee Onboarding & Offboarding System

## Project Overview
HRONE is an automated Employee Onboarding and Offboarding system built on the ServiceNow platform. It helps HR teams manage employee lifecycle processes efficiently by reducing manual tasks, improving transparency, and ensuring smooth operations.

---

## Features

### Onboarding
- Create new employee records
- Automated task assignment to HR, IT, and Admin teams
- Document submission and verification
- Email notifications at each stage
- Role-based access provisioning

### Offboarding
- Employee exit request management
- Asset return tracking (laptop, ID card, etc.)
- Access revocation (email, system access)
- Final settlement workflow
- Exit clearance approvals

---

## User Roles

### Employee
- Submit onboarding details
- Request offboarding
- Track request status

### HR
- Approve onboarding/offboarding requests
- Manage employee records
- Monitor workflows

### IT/Admin
- Provide system access and assets
- Revoke access during offboarding
- Manage assigned tasks

---

## Modules
- Employee Management
- Onboarding Requests
- Offboarding Requests
- Asset Management
- Task Management

---

## Workflow

### Onboarding Process
1. Employee submits details
2. HR reviews and approves
3. Tasks assigned to IT/Admin
4. System access and assets provided
5. Process completed with notification

### Offboarding Process
1. Employee submits exit request
2. HR approval
3. Asset return and access removal
4. Final clearance
5. Process completion

---

## Setup Instructions
1. Log in to your ServiceNow instance
2. Create required tables (Employee, Requests, Assets)
3. Configure Service Catalog items
4. Design workflows using Flow Designer
5. Assign roles and permissions
6. Test onboarding and offboarding processes
